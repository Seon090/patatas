Streams API

La API de Streams permite procesar colecciones de datos de forma funcional (Java 8+).

    .map(n -> n * n)
    .collect(Collectors.toList());

// Sumar todos los n√∫meros

    .filter(n -> n > 5)
    .map(n -> n * 2)
    .reduce(0, Integer::sum);
